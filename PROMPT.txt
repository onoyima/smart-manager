PHASE 2 — WORKER PIPELINE (15 steps, write each as a separate function):

Step 1 — Setup: Pick job from Redis, update job.status = "processing", create /tmp/{job_id}/ working dir
Step 2 — Download: Download S3 original to /tmp/{job_id}/input.mp4, verify MD5 checksum
Step 3 — Audio extraction: FFmpeg extract to 16kHz mono WAV: ffmpeg -i input.mp4 -ar 16000 -ac 1 audio.wav
Step 4 — Transcription: Whisper API on audio.wav, get word-level timestamps, save to DB
Step 5 — Silence detection: librosa detect pauses >0.5s, build silence_windows list
Step 6 — Segmentation: Split into candidate segments by sentence boundaries + silence + 90s max duration
Step 7 — Feature extraction: per segment: energy (RMS), speech_rate (words/sec), keyword_density, sentiment_score (-1 to 1), motion_intensity (frame diff via OpenCV)
Step 8 — Hook scoring: LLM prompt per segment's first 2 sentences: "Rate as social media hook 0.0-1.0. Return only a float."
Step 9 — Viral scoring: ViralScore = (energy*0.30) + (keyword_density*0.20) + (sentiment*0.15) + (speech_rate*0.10) + (motion_intensity*0.10) + (hook_score*0.15). Select top 5 segments.
Step 10 — Clip rendering: FFmpeg trim, silence removal, scale/pad to 9:16 1080x1920, H.264 MP4
Step 11 — Captions: Generate SRT per clip from transcript, burn in with FFmpeg subtitles filter or export .srt
Step 12 — Thumbnails: FFmpeg extract frame at 1 second, save as JPEG
Step 13 — Upload outputs: Upload all MP4s, SRTs, JPEGs to S3 /processed/{job_id}/
Step 14 — Save results: Insert all clip records into DB with scores, S3 keys, durations
Step 15 — Cleanup: Delete /tmp/{job_id}/, update job.status = "done"
Error handling: try/except around full pipeline, log stack trace, update job.status = "failed", trigger retry
